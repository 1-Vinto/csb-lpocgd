Timer para partida de MTG

Inicialmente foi efetuada a construção de um cronômetro, mas para atender a necessidade de alguns amigos, o projeto foi transformado em um timer que limita o tempo de turnos em partidas de Magic the Gathering.(o que aparece na imagem é a ultima versão desenvolvida 1.0.2)
O timer tem tempo programável e a barra de progresso diminui conforme o tempo passa, o projeto foi efetuado em C++.

API cartas de MTG

É basciamente uma API que foi desenvolvida para controle e cadastro de Cartas, As cartas são incluídas por meio do método POST, podem ser editadas por meio do método PUT, deletadas por meio do método DELETE e é possível buscar as cartas por meio do método GET.
A API foi desenvolvida em Python utilizando Flask e TinyDB, as informações que são inseridas na API ficam salvas em um Json.
